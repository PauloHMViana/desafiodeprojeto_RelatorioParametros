# desafiodeprojeto_RelatorioParametros

### Repositorio criado para entrega do desafio de projeto DIO 

**Criando Relatórios Dinâmicos com o uso de Parâmetros no Power BI**

Nele fiz as mudanças proposta pela a especialista de:

- Segunda visão: parâmetros com base em valores.
- Sigam a mesma estilização do relatório.
- Criem uma história para apresentar essa visão sobre os dados.
